File|Open Open an existing QPCR file (Eppendorf only).

File|Open recent files If there are numbered files, these are recently opened files. Choosing one of them will cause it to be opened and to become the current image.

File|Print Print the results.

File|Save Save the current image.

File|Save As Save the current QPCR under a new name, and make the newly named file the current plate.

File|Quit Terminate the program. You will be given the chance to save any unsaved changes.
