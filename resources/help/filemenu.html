File|Open Open an existing QPCR file (for now, Eppendorf only!).

File|Open recent files If there are numbered files, these are recently opened files. Choosing one of them will cause it to be opened and to become the current experiment.

File|Print Print a report with the results. You can choose to print the results table, the standards curves and/or the quantification plots.

File|Export as PDF Export a report in .pdf format with the results. You can choose to export the results table, the standards curves and/or the quantification plots.

File|Save Save the current QPCR experiment.

File|Save As Save the current QPCR experiment under a new name, and make the newly named file the current experiment.

File|Quit Terminate the program. You will be given the chance to save any unsaved changes.
