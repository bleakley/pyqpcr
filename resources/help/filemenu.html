File|Open Open an existing pyQPCR file. This file format is XML based.

File|New project Create a new pyQPCR project. You have to choose the calculation type (relative or absolute), to give a project name and to select your PCR devise. Then, you select which plates (of raw data) you want to import and the path where you want to save your project.

File|Import Add/Import new plate(s) to your existing project. You can import one plate from a raw data file (TXT or CSV file format) given by your QPCR machine or import all the plates of another project (XML) by changing the filter at the bottom of the dialog box. Some samples of such files are given in the sample directory of the source of pyQPCR.

File|Close a plate Remove/Close the current plate from your project.

File|Open recent files If there are numbered files, these are recently opened files. Choosing one of them will cause it to be opened and to become the current experiment.

File|Print Print a report with the results. You can choose to print the results table, the standard curves and/or the quantification plots.

File|Export as PDF Export a report in .pdf format with the results. You can choose to export the results table, the standard curves and/or the quantification plots.

File|Save Save the current QPCR experiment.

File|Save As Save the current QPCR experiment under a new name, and make the newly named file the current experiment.

File|Quit Terminate the program. You will be given the chance to save any unsaved changes.
